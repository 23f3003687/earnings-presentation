Financial Modeling

Sharpe Ratio Analysis:

$$S = \frac{R_p - R_f}{\sigma_p}$$

Where:

  • \(R_p\) = Portfolio return = 15.3%
  • \(R_f\) = Risk-free rate = 4.5%
  • \(\sigma_p\) = Portfolio std deviation = 8.2%

Result: Sharpe Ratio = 1.32 (Excellent risk-adjusted returns)

Our Sharpe ratio of 1.32 indicates excellent risk-adjusted performance. Any ratio above 1.0 is considered good; above 2.0 is very good. We're well-positioned. Contact: 23f3003687@ds.study.iitm.ac.in

Portfolio Valuation

Discounted Cash Flow Model:

$$PV = \sum_{t=1}^{n} \frac{CF_t}{(1+r)^t}$$

Weighted Average Cost of Capital:

$$WACC = \frac{E}{V} \times R_e + \frac{D}{V} \times R_d \times (1-T_c)$$
